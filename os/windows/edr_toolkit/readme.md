This project folder contains specialized toolkits designed for deep visibility, threat hunting, and automated response on Windows systems.

* Kernel-Level Sensor (Rust): A high-performance Windows driver that monitors low-level system activity (syscalls, file I/O, registry) with minimal overhead, providing the raw telemetry needed for deep analysis.
* PowerShell Hunting Toolkit: A specialized script for active threat hunting, focused on uncovering fileless malware, advanced persistence, and stealthy evasion techniques already present in an environment.
* ML-Integrated Sensor (C# & Rust): A real-time monitoring agent that combines memory-native telemetry with a Machine Learning backend to autonomously detect and block zero-day exploits and memory-based attacks.
